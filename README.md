# 🎬 QuickShow - Movie Ticket Booking Application

QuickShow is a full-stack movie ticket booking application built with the **MERN stack**. It allows users to view currently running movies, select and reserve seats, and book tickets with ease. Admins can manage shows, monitor bookings, and analyze ticket sales via a dedicated dashboard.

---

## 🚀 Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Authentication:** Clerk  
- **Event Handling:** Inngest  
- **Movie Data API:** TMDB (The Movie Database API)

---

## ✅ Features

### 👥 User Side
- 🎞️ View currently running movies via TMDB API.
- 🪑 Book specific seats using an interactive seat layout.
- 💳 Option to **pay now** or **pay later**.
- 📩 Clerk-based secure authentication and session handling.

### 🛠️ Admin Dashboard
- ➕ **Add Show** – Create and schedule new movie shows.
- 📋 **List Shows** – View and manage all scheduled shows.
- 📊 **Ticket Analytics** – Visualize and track ticket booking trends.
- 👥 **Customer Booking Records** – View all seat bookings and customer details.
